## [0.1.0] - 2025-02-15

### Added

- Initial skeleton: properties, units, parties, contracts, billing (Invoice, Payment)
- Django 5 + DRF backend with JWT authentication
- React + TypeScript frontend (Vite) with login and property list
- Docker Compose for self-hosted deployment
- BSL license with $100K revenue threshold
- CLA for contributors
- OpenAPI/Swagger documentation
- CI workflow (pytest, black, isort, flake8, eslint, prettier)
- React Parties and Contracts pages (list + create)
- Unit entity and API for contract form
- E2E tests (Playwright) for core flow
- Notifications: email on contract creation (Phase 2 start)
